欢迎访问如何使用 Read the Docs!
============================

Read the Docs 是一个文档编写工具, 它简化了文档编写, 并提供了
自动化构建,版本控制, 以及网站寄存。

本文档致于介绍Read the Docs的基础使用, 以及对本文档的编写。

更多详情, 请参考 :doc:`usage` 章节，

以及 :ref:`installation` the project.
